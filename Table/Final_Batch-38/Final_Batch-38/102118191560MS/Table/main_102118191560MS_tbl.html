Recent efforts have been focused on increasing cluster stimulation efficiency via implementing diversion agents. Diversion agents, such as bio balls/sealers and biodegradable particulates, when coupled with limited entry techniques, allows for increasing stage length with decreased cluster spacing, while maintaining cluster stimulation efficiency.
