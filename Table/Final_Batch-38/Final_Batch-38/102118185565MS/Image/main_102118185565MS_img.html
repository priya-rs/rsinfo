A set of 391 fields distributed in the 8 more relevant producing oil basins in Colombia were selected for setting up the group of elements for developing the proposed correlation (Figure 1). Subsequently, the fields were handpicked contemplating the fact that the cluster had to surpass 50[%] of the basin production. The distribution of the 51 fields is shown in Table 1, where it can be seen that the total production from them, represents 71.57[%] of 1’020.325, 029 [Bbl/d], which is the Colombia's daily production (ANH, 2016).
