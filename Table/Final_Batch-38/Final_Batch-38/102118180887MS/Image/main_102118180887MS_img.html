BHP Billiton’s Greater Angostura field discovered in 1999 is located in Block 2 (c) 25 miles from the coast of Trinidad and Tobago. The field was developed in 3 phases: Phase 1, which includes the Kairi, Canteen and Horsts fields; Phase 2, which includes the Aripo gas development; and Phase 3 which is scheduled to be completed by the second half of 2016. The application improves production surveillance capabilities as potential issues can be identified more readily. A graphical user interface –as shown in the example in Fig. 6– was developed to display the parameters calculated by the production model as well as the values retrieved from the real-time data acquisition system. Discrepancies between model-calculated and actual parameters alert the engineer to focus on a specific area of the system when performing the necessary diagnostics.
