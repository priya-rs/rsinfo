An example of some pre-determined thresholds for field-level history-match quality is shown in Table 2. The thresholds are often defined based on the metering and gauge quality. For example, if the water meters are less accurate, greater leniency can be assigned for this metric's threshold. It is also not mandatory to set a threshold for a given metric, such as for the FBHP metric (i.e. N/A or not assessed), if the well does not have an existing gauge.
