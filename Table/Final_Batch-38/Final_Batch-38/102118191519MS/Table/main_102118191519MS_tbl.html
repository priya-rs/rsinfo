Grid cluster can be used in case of a set of loosely bound tasks is considered (e.g. multivariate reservoir simulations), so it is great for optimization of well placement and settings configuration and history matching. The system has been applied for wide range of reservoir simulation cases; linear performance improvement according to the number of nodes was confirmed (see Table 1, moderate variations occur due to daily user’s workload). However, such system is not capable of solving large problems that do not fit one node, e.g. simulation of high-dimentional reservoir models.
