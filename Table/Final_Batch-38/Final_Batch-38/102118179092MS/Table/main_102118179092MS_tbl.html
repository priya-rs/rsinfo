The deployment was performed in seven runs. Table 3 outlines the functions of the main components of the tool string for these runs. Fig. 6 and Fig. 7 show a schematic of the deployment string, which includes the locations of the main components.

Table 3 Main components of deployment system.

Table 3 Main components of deployment system.

Component Description/Function
Swivel Prevents the tool string from rotating
Release system Emergency release system, for controlled release and retrieval of the tool string. It is activated by surface command or via time delay memory mode. Proper placement of the emergency release in the tool string enables more efficient retrieval/fishing
Casing collar locator (CCL) Enables precise positioning of the straddle, anchor and blank pipe
Setting tool A device for anchoring and setting plugs and packers in the wellbore
Tractor Conveys the tool string to the required position in the well and also tractors the stinger seal of the component to latch into the seal bore
Stroker Powered mechanical device that anchors in the tubing and can stroke a piston bi-directionally and repeatedly, with real time control and monitoring of the applied force and movement of the piston
4” GS Pulling tool A slickline running tool designed to deploy and retrieve downhole components with an internal fishing neck. It can be released by jarring down with the slickline and breaking some shear pins, which allow it to come off the fishing neck. In this job, where it was not possible to deploy the system on slickline due to the high deviation (72°) the release was effected by the stroker.
