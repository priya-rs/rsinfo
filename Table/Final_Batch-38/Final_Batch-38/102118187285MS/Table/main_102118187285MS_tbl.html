To validate the well testing model proposed in this paper, an MFHW is numerically simulated by use of KAPPA’s Saphir software (Essca 2011) to compare the numerical result with the analytical one. The model proposed in this paper is simplified to conform to the numerical well-testing model, and the parameters are given in Table 1. The horizontal well is assumed to be uniformly intercepted by five vertical hydraulic fractures with same properties (e.g. half-length, width, height, flux, infinite conductivity).

Finally, we compare the results from the well testing and ERT method with those from PLT to validate the reliability of the method proposed in this paper. The total flow rates are not the same because production profile test is earlier than pressure test for about one month. First, we calculate the contribution ratio of each combined segment for two kinds of results according to the data from Table 4 and Table 5 to keep the same standard, shown in Table 6. The results in the pink region have the same trends with those in the blue region, which demonstrates the accuracy of the well testing method in this paper.
