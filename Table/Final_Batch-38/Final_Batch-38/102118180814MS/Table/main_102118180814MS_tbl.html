From the simulation results summarized in Table 1 it is possible to identify the efficiency of the waterflooding for each layer. According to the (dimensionless) Pore Volumes Injected (VPI, Spanish acronym) a typical pattern has a value of under 0.3. A simple Buckley-Leveret assessment based on typical viscosity values that this value has to be significantly increased towards 1.2, why an improvement of the vertical efficiency must be overcome. In the fault blocks with best production performance such as blocks 3, 4, and 5, a higher pore volume injection has been reached and the VPI has been estimated to be at around 0.7 to above 1.
