We used a Carter-Tracy analytical model to represent the aquifer. This model uses tables of dimensionless time versus a dimensionless pressure influence function. In this case, the table was for an infinite aquifer with constant terminal rate as given by Van Everdingen and Hurst. Table 1, present the final parameters used in the history match and we see in Fig. 4 the aquifer – reservoir cell connections.
