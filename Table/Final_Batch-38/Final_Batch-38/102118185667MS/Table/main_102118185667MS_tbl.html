A deconvolution algorithm was applied to the acoustic power of each FCD to find the frequency characterization of each nozzle size. Table 1 shows the harmonic frequencies of three different nozzle sizes that were used in well "A". Similar frequencies were also observed in well "B". The acoustic harmonic frequencies observed for each nozzle changes slightly for different flow condition. This harmonic characterization can be used to determine if the installed nozzles are in the critical flow and the nozzle open flow area is consistent with the design. If nozzle plugging was occurring, this effect could be detected.
