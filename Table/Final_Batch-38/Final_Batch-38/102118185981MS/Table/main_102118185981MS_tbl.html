In this study, according to the main diagenetic types and strength controlling the reservoir quality to name different diaganetic facies, the diagenesis which have tiny influence are not involved in naming. When there are more than 2 kinds of main diagenesis, we should use compound nomenclature. The data of ACR (Apparent Compaction Ratio) and ADR (Apparent Dissolution Ratio) are listed in Table 1. By using these parameters, we can establish a standard, and divide the diagenetic facies into six types.
